- Project root: `/home/j3su5/learning/mike_shirts/`
- Django project name: `mike_shirts` (CONTEXT.md decision)
- Single app `accounts` handles User, profiles, addresses (CONTEXT.md decision)
- AbstractUser extended with: phone, rfc (optional), auth_provider (CONTEXT.md + decision journal)
- Django Groups for roles: ventas, envios, produccion, admin_full (CONTEXT.md decision)
- Proxy models: Customer (no staff groups), StaffUser (any staff group) (CONTEXT.md decision)
