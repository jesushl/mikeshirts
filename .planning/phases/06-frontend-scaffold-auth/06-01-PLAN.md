# Plan 06-01 — Frontend Scaffold & Auth UI

## Status: COMPLETE

## Scope

Build the React SPA scaffold with authentication UI and account management.

## Tasks

### Wave 1 — Scaffold
- [x] Vite 6 + React 18 + TypeScript project setup
- [x] Tailwind CSS 4 via @tailwindcss/vite plugin
- [x] Brand color palette as CSS custom properties
- [x] Inter font from Google Fonts
- [x] React Router 6 with all route placeholders
- [x] Zustand stores (authStore, cartStore)
- [x] Axios API client with JWT interceptor (auto-refresh + queue)
- [x] Logo: calavera de ratón estilo mexicano Día de Muertos
- [x] Vite proxy /api → localhost:8000

### Wave 2 — Auth UI
- [x] Layout: Header (responsive + mobile menu), Footer, Outlet wrapper
- [x] HomePage: hero section with brand identity
- [x] LoginPage: email/password form with error handling
- [x] RegisterPage: full registration form with field-level validation
- [x] ProfilePage: edit profile, list/delete addresses, logout
- [x] NewAddressPage: full address form with is_default toggle
