# Plan 07-02 — Página de Producto

## Objective

Build the product detail page with image gallery, variant selection (size + color), size chart, availability display, and add-to-cart functionality. Track `product_view` event.

## Context

- `fetchProduct(slug)` already defined in catalog service (from 07-01).
- `ProductDetail` type has: images[], variants[], size_chart[], category, description, base_price.
- Each Variant has: id, size, size_display, color, color_display, sku, effective_price, stock (quantity, production_days, availability, is_low_stock).
- Cart API: `POST /api/cart/items/ {variant_id, quantity}` — adds to cart, returns full Cart.
- Analytics: `POST /api/analytics/event/ {event_type: 'product_view', metadata: {product_id, slug}}`.
- Color map for swatches: negro→#0a0a0a, blanco→#f5f5f5, gris→#6b7280, azul_marino→#1e3a5f, rojo→#e11d48, verde→#16a34a.
- Size order: XS, S, M, L, XL, XXL.
