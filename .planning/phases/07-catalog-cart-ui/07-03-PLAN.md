# Plan 07-03 — Carrito + Home Destacados

## Objective

Build the CartDrawer (slide-over) and CartPage (full page), connect cartStore to real API, merge session cart on login, add featured products to HomePage, and wire remaining analytics.

## Context

- Cart API endpoints:
  - `GET /api/cart/` → Cart (items, item_count, subtotal, shipping, total)
  - `POST /api/cart/items/ {variant_id, quantity}` → Cart
  - `PATCH /api/cart/items/<variant_id>/ {quantity}` → Cart
  - `DELETE /api/cart/items/<variant_id>/delete/` → 204
  - `DELETE /api/cart/` → 204 (clear cart)
  - `POST /api/cart/merge/` → Cart (auth required)
- Cart response includes: items[] (variant_id, product_name, product_slug, size, color, sku, unit_price, quantity, line_total, image), item_count, subtotal, shipping ($249), total.
- Current cartStore has: cart, isOpen, setCart, openCart, closeCart, toggleCart.
- Header already has cart icon with badge (item_count) and toggleCart on click.
- useAuth.login() sets auth — need to trigger cart merge after login.
- `fetchProducts({featured: true, page_size: 8})` gets featured products for HomePage.
- Shipping is flat $249 MXN (from backend settings, returned in cart response).
