# Features Research — Mike Shirts

## Table Stakes (el cliente los espera, sin ellos no hay confianza)

| Feature | Descripción |
|---------|-------------|
| Catálogo con fotos de calidad | Imágenes múltiples por producto, zoom, vista en modelo |
| Filtros y búsqueda | Por talla, color, precio, categoría temática |
| Tabla de tallas | Con medidas reales en cm; reduce devoluciones 30%+ |
| Carrito persistente | Sobrevive refresh y login/logout |
| Checkout en <3 pasos | Dirección → método de pago → confirmación |
| Confirmación por email | Resumen de compra + número de pedido |
| Responsive/mobile-first | 70%+ del tráfico e-commerce en México es móvil |
| HTTPS y datos seguros | Obligatorio para pagos y confianza del usuario |
| Políticas claras | Envío, devoluciones, privacidad visibles |
| Seguimiento de pedido | Estado básico: pagado → en producción → enviado → entregado |

## Diferenciadores (dan ventaja competitiva)

| Feature | Descripción |
|---------|-------------|
| Formatos Meta-ready | Imágenes de producto en 1:1, 4:5, 9:16 listas para publicar |
| Landing por diseño | Cada diseño tiene su URL shareable para redes sociales |
| Analítica de fuente | Saber si la venta vino de Instagram, Facebook, directo, etc. |
| Panel admin por roles | Cada rol ve solo lo que necesita (ventas ve pedidos, producción ve tickets) |
| Tickets de producción | Workflow interno: stock bajo → ticket automático → producción → restock |
| Funnel de conversión | Visita → producto → carrito → checkout → compra (métricas por paso) |
| Stock híbrido | Inventario fijo + on-demand sin fricción visible para el cliente |

## Anti-features (NO construir en v1)

| Feature | Razón |
|---------|-------|
| Wishlist / favoritos | Nice-to-have, no crítico para primeras ventas |
| Reviews/ratings de clientes | Negocio nuevo, no hay masa crítica de reviews |
| Chat en vivo | Complejidad operativa sin equipo de soporte dedicado |
| Programa de lealtad/puntos | Prematuro sin base de clientes recurrentes |
| Comparador de productos | No aplica bien para playeras (no son specs técnicos) |
| Multi-idioma | Solo México, solo español |
| Suscripción/caja mensual | Modelo de negocio diferente, no es el objetivo |
